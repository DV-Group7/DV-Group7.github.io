GRAPH I - SANKEY GRAPH

A Sankey diagram is a visualization used to depict a flow from one set of values to another, in which the width of each flow pictured is based on its quantity.
In this specific case there is a double alluvial diagram: the first one has as input the Carbon storage and as output of the top 10 tree species in Trento. Whereas the input of the second one is the top 10 tree species in Trento and its output is the total annual benefit per year in euros. Finally a tooltip provides more detail about the carbon storage of each tree and its annual benefit per year.
